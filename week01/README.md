# 1주차 과제. CLI 프로그램 제작

## 프로그램 소개
csv 파일에 저장된 데이터의 도서들을 사용자의 이름을 기준으로 대여하거나 반납할 수 있는 CLI 프로그램입니다.
대여 기간은 2주일이며, 대여 기간 이후 반납하고자 한다면 연체한 기간과 관계 없이 연체료를 100원 내야 합니다.

! 프로그램을 정상적으로 종료할 때만 데이터가 저장됩니다.
